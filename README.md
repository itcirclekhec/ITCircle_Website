## 🎯 What is IT Circle?

**IT Circle** is the vibrant Information Technology community at **Khwopa Engineering College** - a full-stack web platform that serves as the digital heart of our tech community.

### 🌟 Our Mission
> *"Empowering students through technology, fostering innovation, and building the next generation of tech leaders in Nepal."*

### 🎯 What We Do
- **🏆 Showcase Student Projects** - Highlight innovative work from our talented students
- **📅 Host Tech Events** - Workshops, seminars, hackathons, and networking sessions
- **🌐 Build Community** - Connect students, alumni, and industry professionals
- **📚 Share Knowledge** - Resources, tutorials, and learning materials
- **🎓 Career Development** - Internship opportunities and industry connections

## ✨ Key Features

### 🚀 **Modern Tech Stack**
- **Frontend**: Next.js 15 + TypeScript + Tailwind CSS
- **Backend**: Django 5 + PostgreSQL + REST API

## 🚀 Quick Start

### Prerequisites
- **Node.js** 18+ • **Python** 3.11+ • **PostgreSQL** 13+ • **Git**

### One-Command Setup
```bash
# 1. Clone the repository
git clone https://github.com/KhEC-IT-Circle/ITCircle_Website.git
cd ITCircle_Website

# 2. Start the backend
cd backend && python -m venv venv && source venv/bin/activate
pip install -r requirements.txt && python manage.py runserver

# 3. Start the frontend (new terminal)
cd frontend && npm install && npm run dev

# 4. Open http://localhost:3000 🎉
```

> **Need help?** Check our [📖 Detailed Setup Guide](./docs/development/README.md)
